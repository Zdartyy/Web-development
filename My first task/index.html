HTML & CSS Basics Summary

HTML Summary

HTML (HyperText Markup language) is used to define our page conent, structure and meaning. You don't use it for styling purposes. Use CSS for that instead!

  1. HTML uses elements to describe (annotate) content
  2. HTML elements typically have an opening tag and a closing tag
  3. You can also have void (empty) elements like images
  4. You can also configure elements using attributes
  5. There are several types of elements: inline, block, inline-block

CSS Summary

CSS (Cascading Style Sheets) is used for styling your page content
